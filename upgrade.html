Upgrade!

About

Date: May-July 2016

Upgrade! is a 2D platformer I made between semesters at uni. The core mechanic for Upgrade! came from a idea I had a while ago for a game that evolved to suit your playstyle. For example, if you spammed the attack key, you would be able to attack faster, and more often; if you moved around a lot, you would get mobility increases, like movement speed. Although that idea wasn't fully brought to fruition in this game, Upgrade! demonstrates the base functionality of that mechanic, and hopefully I can revisit it at a later date, to see how far I can push the idea.
